- **API key parameter name**: APP-AUTH
- **Location**: HTTP header

Example

```go

		context.Background(),
		chapar.ContextAPIKeys,
		map[string]chapar.APIKey{

		},
	)
r, err := client.Service.Operation(auth, args)
```
